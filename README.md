# 💪 Treino da Mãe App

Aplicação web simples para acompanhamento de treinos, com foco em usabilidade mobile e visualização fácil de exercícios através de vídeos.

---

## 🚀 Funcionalidades

- Separação de treinos (A, B, C, D)
- Vídeos demonstrativos de cada exercício
- Layout responsivo (funciona no celular)
- Interface simples e intuitiva
- Navegação por abas
